# 🌦️ React Weather App

A playful and animated weather application built with **React** and the **OpenWeatherMap API**. Users can enter any city to retrieve and view current weather conditions, including temperature, description, and icon representation.

---

## 🚀 Features

- 🔍 Search by city name
- 🌡️ Displays temperature in Celsius
- 🌥️ Shows weather description & icon
- ✨ Playful, animated UI with floating clouds
- 📱 Fully responsive and mobile-friendly
- ⚛️ Built using React with functional components & hooks
- 🔐 Environment variable support for API key security

---

## 🔧 Tech Stack

- **React**
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **OpenWeatherMap API**
